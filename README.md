# CEAT SHOPPEE RG TYRES

Static showroom website prepared for GitHub Pages publishing.

## Files

- `index.html`
- `styles.css`
- `script.js`

## Publish On GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js`.
3. Open the repository on GitHub.
4. Go to `Settings` -> `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select branch `main` and folder `/root`.
7. Save and wait for the site URL to appear.

## Notes

- This version is intentionally static so it works cleanly on GitHub Pages.
- Customer feedback is routed through Google Maps reviews or email instead of browser-only storage.
- If you later want professional online forms, bookings, or admin tools, add a backend such as Firebase or Supabase.
